# Marcus Chen - Synthetic Identity Persona

A fictional "perfect candidate" who passes all traditional hiring checks — used to demonstrate how synthetic identities infiltrate organizations.

## Story

Marcus Chen appears to be an ideal senior engineer candidate:
- Strong resume with reputable companies
- Active GitHub with contributions
- Professional LinkedIn presence
- Clean background check
- References that check out

But he doesn't exist. He's a synthetic identity created to gain access to sensitive systems.
